Building a Spam Filter with Naive Bayes¶

In this project a spam filter is built based on the Naive Bayes algorithm. It will be used to classify SMS messages as spam or non-spam. The spam-filter is being trained with a dataset of 5572 SMS messages that are already classified by humans (The UCI Machine Learning Repository) (https://archive.ics.uci.edu/ml/datasets/sms+spam+collection) See wikipedia for background information (https://en.wikipedia.org/wiki/Naive_Bayes_classifier)
